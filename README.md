# rock-paper-scissors
The Odin Project - Rock Paper Scissors

Assignment (Part 1): Write a JavaScript program that allows a user to play a five-round game of rock paper scissors against the computer in the developer console (no GUI).

Assignment (Part 2): Update the JavaScript to play the game with
a GUI.

Concepts:
- Basic JavaScript
- DOM Manipulation/Events
- Git Branching
